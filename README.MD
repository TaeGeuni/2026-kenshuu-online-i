# 2026 東京ITスクール online-i プログラミング研修記録

はじめまして！2026年 東京ITスクール online-iクラスのプログラミング研修で、毎日の学習内容をまとめ、同期の皆さんと知識を共有するために作成したリポジトリです。

## 📖 リポジトリの目的
* **TIL (Today I Learned):** 毎日新しく学んだこと、躓いたポイントやその解決過程を記録します。
* **知識の共有:** 自分だけが知っているのはもったいないTipsや、役立つ資料を共有します。
* **コミュニケーション:** 授業内容で理解できなかった部分や、より良いコードの書き方について、皆さんと自由に意見交換をしたいです。

---

## 💬 意見交換・ディスカッションへの参加方法

このリポジトリは私個人のメモ帳だけでなく、**online-iクラスの皆さんと一緒に作り上げていく空間**にしたいと思っています。

意見を交わしたい、質問したいという方は、ぜひ以下の方法を活用してください！

1. **GitHub Issues の活用**
   * 特定のコードに関する質問や、授業内容で議論したいテーマがあれば、`Issues`タブに新しくissueを作成してください。
2. **GitHub Discussions (推奨！)**
   * 雑談、情報共有、Q&Aには、`Discussions`タブを活用してください。
3. **Pull Request (PR) 大歓迎**
   * 私がまとめた内容に間違いがあったり、追加したい有益な情報があれば、いつでもPRを送ってください！
